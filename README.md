# SNMP IF-MIB Interface Monitor

CLI-утилита на Go для отображения статуса сетевых интерфейсов и количества трафика на них через SNMP IF-MIB в реальном времени.

## Возможности

- Получение списка сетевых интерфейсов через SNMP
- Отображение статуса интерфейса: up/down
- Отображение входящего и исходящего трафика
- Расчет текущей скорости в Mbps
- Поддержка SNMP v2c

## Используемые OID

- ifName: 1.3.6.1.2.1.31.1.1.1.1
- ifOperStatus: 1.3.6.1.2.1.2.2.1.8
- ifHCInOctets: 1.3.6.1.2.1.31.1.1.1.6
- ifHCOutOctets: 1.3.6.1.2.1.31.1.1.1.10
